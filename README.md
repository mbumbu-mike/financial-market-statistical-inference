# Financial Market Statistical Inference

## Project Overview

This project applies statistical inference techniques to a real-world financial market dataset obtained from Kaggle.

The analysis covers:

- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Confidence Intervals
- Parametric Statistical Methods
- Nonparametric Statistical Methods
- Correlation Analysis
- Regression Analysis
- Statistical Interpretation and Reporting

## Dataset

World Stock Prices (Daily Updating)

Source:
https://www.kaggle.com/datasets/nelgiriyewithana/world-stock-prices-daily-updating
